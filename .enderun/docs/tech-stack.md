# Technology Stack

## Core
- **Frontend:** React 19 + Vite (SPA)
- **State Management:** Zustand
- **Styling:** Panda CSS (Zero UI Library Policy)
- **Backend:** Node.js 20+ + Fastify
- **Database:** PostgreSQL (Kysely)
- **Types:** TypeScript (Branded Types)

## Tools
- **Testing:** Vitest
- **Logging:** Pino
- **Orchestration:** AI-Enderun
